Evidenčná kniha vystavených WooCommerce faktúr pod menu PDF Invoices s filtrom a exportom CSV.

== Description ==
Samostatný AR Design modul pre evidenciu vystavených faktúr z WooCommerce PDF Invoices & Packing Slips.

Funkcie:
* Evidenčná kniha faktúr v administrácii
* Filtre podľa stavu, dátumu, zákazníka a čísla faktúry
* Export podľa filtra do CSV
* Predvoľba export profilu pre ekonomický SW

== Installation ==
1. Nahrajte plugin do `wp-content/plugins/ar-design-invoice-ledger`
2. Aktivujte plugin v WordPress administrácii
3. Otvorte WooCommerce -> PDF Invoices -> Evidenčná kniha faktúr
