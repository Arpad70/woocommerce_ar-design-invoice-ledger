# Changelog

## 1.0.3
- Added robust admin menu parent fallback so the Invoice ledger page appears under PDF Invoices when available, or under WooCommerce otherwise.

## 1.0.2
- Added WooCommerce HPOS compatibility declaration (`custom_order_tables`) to remove incompatible plugin warning in WooCommerce features screen.

## 1.0.1
- Added complete localization support with translation files for Slovak, Czech, and English (`.po` + `.mo`).
- Normalized plugin translatable source strings for full i18n coverage across admin UI and export labels.

## 1.0.0
- Initial standalone AR Design Invoice Ledger module.
- Added admin submenu under PDF Invoices with invoice ledger table.
- Added filters by order status, date range, customer, and invoice number.
- Added CSV export with software profile selector.
- Added GitHub release updater integration and automated release workflow.
